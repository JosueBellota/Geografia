# Geografía - Cartografía y Escalas

## Project Overview

This is an educational repository focused on geography, specifically cartography (map-making) and scales. The collection contains solved exercises, proposed problems, examinations, and interactive HTML solutions related to geographical calculations and cartographic techniques.

The main topics covered include:
- Coordinate calculations (UTM coordinates)
- Distance calculations between points
- Azimuth and orientation calculations
- Scale calculations on maps
- Topographic calculations (elevation differences, slopes)
- Area calculations using coordinate geometry
- Geodesic distance calculations between geographical coordinates

## Key Mathematical Concepts

The exercises involve several mathematical concepts:
- Euclidean distance calculations: √((X₂ - X₁)² + (Y₂ - Y₁)²)
- Trigonometric calculations for azimuth using atan2 function
- Scale conversions: Real distance = Map distance × Scale denominator
- Slope calculations: Percentage and angular measurements
- Coordinate geometry for area calculations (Gauss's area formula)
- Spherical trigonometry for geodesic calculations
